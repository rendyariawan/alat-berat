# Bangka Heavy Rental — Company Profile

Landing page company profile untuk usaha sewa alat berat. Dibuat menggunakan HTML dan CSS murni sehingga ringan, mudah dipasang, dan tidak membutuhkan proses build.

## Fitur

- Desain industrial premium dengan aksen oranye
- Responsif untuk desktop, tablet, dan perangkat mobile
- Navigasi mobile tanpa JavaScript
- Hero section dan tombol WhatsApp
- Statistik perusahaan
- Profil dan keunggulan perusahaan
- Katalog armada alat berat
- Layanan, portofolio proyek, dan alur pemesanan
- Formulir permintaan penawaran
- Optimasi dasar SEO dan aksesibilitas

## Menjalankan

Buka `index.html` langsung menggunakan browser atau jalankan melalui ekstensi Live Server di Visual Studio Code.

## Data yang Perlu Disesuaikan

Cari dan ganti data contoh berikut pada `index.html`:

- `Bangka Heavy Rental` — nama perusahaan
- `6281234567890` — nomor WhatsApp tanpa tanda `+`
- `+62 812-3456-7890` — nomor yang ditampilkan
- `info@bangkaheavyrental.co.id` — alamat email
- `Pangkalpinang, Bangka Belitung` — alamat/lokasi
- Statistik perusahaan dan informasi unit

Gambar saat ini menggunakan URL eksternal Unsplash. Untuk produksi, sebaiknya unduh foto milik perusahaan ke folder `assets/images` lalu ubah sumber gambar pada `index.html` dan `styles.css`.
